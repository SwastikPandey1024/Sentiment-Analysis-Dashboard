# 📊 Sentiment Analysis Dashboard (Power BI)

## 🚀 Overview
This project presents an interactive **Power BI dashboard** built on sentiment analysis data to extract meaningful insights from textual datasets. It combines **data analytics, visualization, and business intelligence** to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sentiment trends (Positive, Negative, Neutral)
- Visualize user behavior and feedback patterns
- Enable real-time insights through interactive dashboards
- Support business strategies using data storytelling

---

## 📌 Features
- 📈 Interactive Power BI Dashboard
- 🔍 Sentiment Distribution Analysis
- 📊 Trend Analysis over time
- 🎯 Category-wise sentiment breakdown
- ⚡ Dynamic filters and slicers for better exploration

---

## 🛠 Tools & Technologies
- Power BI  
- Python (for preprocessing & sentiment analysis)  
- Pandas, NumPy  
- NLP (Natural Language Processing)
